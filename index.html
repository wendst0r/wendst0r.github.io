Проблема состояния водных объектов - основная экологическая проблема всей планеты Земля. Основная составляющая мусора - пластиковые бутылки, стаканчики, жестяные банки и т.д. Лёгкий по весу, этот мусор долго остаётся на поверхности.
Наш робот - новый прорыв в сфере очистки поверхности прудов, водохранилищ, озер и других небольших водоёмов. Наша ценовая политика позволяет использовать нашего робота как для использования в личных целях, так и для борьбы с крупными экологическими проблемами.
    Устройство работает следующим образом:
  • Робот ставится на воду и включается при помощи пульта
  • Устройство движется по водной глади, собирая мусор в контейнер с помощью специального конвейера
  • Как только контейнер для сбора мусора наполнится, робот возвращается на то место, откуда его запустили, с помощью GPS
  • Затем мусор перерабатывается
  • Робот делает так до тех пор, пока вся поверхность водоёма не будет очищена
